#include <stdio.h>
void main()
{
float a = 0.5;
double b = 1.2;
int c = 3;
b = b + a + c;
/* 輸出 a, b, c 到螢幕 */
printf( " a = %3.1f, b = %3.1f, c = %d ", a ,b, c );
}

  • float a = 0.5;: 宣告一個浮點數變數 a 並賦值為 0.5。

  • double b = 1.2;: 宣告一個雙精度浮點數變數 b 並賦值為 1.2。

  • int c = 3;: 宣告一個整數變數 c 並賦值為 3。

  • b = b + a + c;: 將 b 的值加上 a 和 c 的值，然後將結果賦值給 b。

  • printf(" a = %3.1f, b = %3.1f, c = %d ", a, b, c);: 使用 printf 函式將變數 a、b、c 的值輸出到螢幕上，其中 %3.1f 表示浮點數輸出，保留一位小數，並總寬度為3；%d 表示整數輸出。
